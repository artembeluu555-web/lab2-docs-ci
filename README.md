#  Система бібліотеки — Документація та CI/CD (Лабораторна робота №2)

Цей репозиторій містить реалізацію **лабораторної роботи №2**, яка є продовженням попередньої.  
Метою роботи є:

- додати коментарі **Doxygen** у вихідний код;
- створити файл **Doxyfile**;
- налаштувати **GitHub Actions** для автоматичної генерації документації;
- автоматично публікувати документацію через **GitHub Pages**.

У програмі реалізовано об’єктно-орієнтовану модель бібліотеки з такими класами:

- `Book`, `PrintedBook`, `EBook`, `AudioBook`
- `User`, `Student`, `Librarian`
- `Catalog`
- `Library`

---


##  CI/CD (GitHub Actions)

Workflow виконує такі кроки:

1. Завантаження репозиторію  
2. Встановлення Doxygen  
3. Генерація HTML-документації  
4. Створення артефакту `docs/html`  
5. Автоматичний деплой у гілку `gh-pages`

---


## ✔ Висновок

Під час виконання лабораторної роботи було виконано:

- документування коду за допомогою Doxygen;
- генерацію HTML-документації;
- налаштування CI/CD;
- автоматичний деплой документації на GitHub Pages.

Документаційна система працює повністю автоматично.
